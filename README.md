# 🌐 IP Address Logger & Geolocation

Este proyecto es una herramienta de auditoría de red diseñada para capturar la dirección IP pública de un dispositivo y obtener su información geográfica aproximada (País, Ciudad, ISP) de forma silenciosa.

## ✨ Características
- **Captura Silenciosa**: No requiere permisos de GPS por parte del usuario.
- **Geolocalización por IP**: Utiliza la base de datos de IP-API para localizar el nodo de conexión.
- **Redirección Automática**: Una vez capturada la información, el usuario es redirigido a una URL segura para evitar sospechas.
